## Plánované vstupy a výstupy
- `clk`: Hlavní hodiny
- `rst`: Reset stopek
- `BTNU`: Start/Stop
- `BTNC`: Uložení mezičasu
- `BTNR`: Listování v mezičasech
- `BTNL`: Zobrazení mezičasu
- `CA-CG, DP`: Segmenty displeje
- `AN`: Anody displeje
